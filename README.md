# 📚 Library Management System

A simple **console-based Library Management System** built using **Java**.
This project demonstrates the use of **Object-Oriented Programming (OOP)** concepts and **File Handling** to manage books and students in a library.

## 🚀 Features

* Add new books to the library
* Register new students
* Issue books to students
* Return issued books
* View all books in the library
* View all registered students
* Data is stored using file handling so it persists after program restart

## 🛠️ Technologies Used

* Java
* OOP Concepts (Classes, Objects, Encapsulation)
* File Handling
* ArrayList
* Scanner for user input

## 📂 Project Structure

* `Book` class → Stores book information
* `Student` class → Stores student details and issued books
* `LibraryManagementSystem` → Main class handling system logic

## ▶️ How to Run

1. Clone the repository
2. Open the project in any Java IDE
3. Compile and run the `LibraryManagementSystem.java` file
4. Use the menu options to manage books and students
